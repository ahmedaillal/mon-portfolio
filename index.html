01

Atlas Climatique Interactif du Maroc

Application web cartographique permettant la visualisation dynamique des données de température à travers le Maroc. Les données climatiques sont extraites automatiquement depuis la plateforme ERA5 (Copernicus) via une pipeline Python, puis représentées sur une carte Leaflet enrichie de filtres temporels et de couches thématiques interactives.

Python Leaflet.js ERA5 / Copernicus NumPy GeoJSON

Automatisation de la Symbolisation Cartographique sous QGIS

Intégration de scripts Python dans QGIS pour automatiser et personnaliser la symbolisation de fichiers Shapefile. Cette solution standardise les rendus cartographiques à grande échelle, réduit les délais de production et garantit une cohérence visuelle sur l'ensemble des livrables SIG.

QGIS Python Shapefile Automatisation SIG

03

Serveur WPS Géospatial — Traitements Spatiaux & Statistiques

Déploiement d'un serveur WPS (Web Processing Service) exposant plusieurs processus OGC standardisés : calcul automatisé de superficies agricoles, analyses de distributions spatiales et extraction de moyennes climatiques ERA5 sur des séries temporelles. Une architecture orientée interopérabilité pour l'infrastructure de données spatiales.

PyWPS 52°North WPS OGC Standards ERA5 Python
↗
04

Détection de Serres par Deep Learning — Souss-Massa

Modèle de segmentation sémantique appliqué à la détection automatique des serres agricoles dans la région de Souss-Massa à partir d'images satellitaires haute résolution. Ce projet permet une cartographie précise de l'expansion des cultures sous abris — un enjeu clé pour la gestion des ressources en eau.

Deep Learning Segmentation sémantique Python Télédétection
↗
05

Classification de la Qualité de l'Eau via Google Earth Engine

Exploitation de la synergie entre télédétection multispectrale et intelligence artificielle via Google Earth Engine pour évaluer la qualité de l'eau de différents plans d'eau. L'approche combine des indices spectraux (turbidité, chlorophylle) avec des algorithmes de classification supervisée, offrant un outil de suivi environnemental scalable.

Google Earth Engine Machine Learning Télédétection Indices spectraux
